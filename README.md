# cli

Command Line Interface tool for developing, building and deploying Elvia applications.

## Installation

Requires [Go](https://golang.org).

### Install from GitHub

```bash
go install github.com/3lvia/cli/cmd/3lv@latest
```

### Install from source

```bash
git clone git@github.com:3lvia/cli.git
cd cli
go install ./3lv/cli
```

## Usage

```bash
3lv --help
```
